## [1.0.0] - 2025-01-XX

### Added
- Implementação inicial do sistema de detecção de fraudes em documentos
- Integração com Azure AI Document Intelligence (Form Recognizer)
- Modelo de Machine Learning usando Random Forest
- Detecção de QR Codes para verificação de autenticidade
- Extração de características de documentos
- Sistema de logging em CSV para auditoria
- Notebook Jupyter com exemplo de uso
- Documentação básica do projeto

### Features
- Extração de texto de documentos usando Azure AI
- Detecção automática de QR Codes
- Classificação de documentos como autênticos ou suspeitos
- Score de fraude (0-1)
- Logs detalhados para compliance
- Suporte a múltiplos formatos de imagem
